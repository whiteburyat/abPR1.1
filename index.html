Знакомство с frontend разработкой

Сайт существует для ознакомления с темой разработки и дизайна сайтов.

HTML5 - HyperText Markup Language - Внедренный в 2013 году язык для структурирования и презентации содержимого сайтов во всемирной паутине. Пятая версия html была разработана как для читаемости кода человеком, так и для улучшения работы с мульти-медиа техлогиями с сохранением обратной совмeстимости

XMLHTML5 - eXtensible Markup Language - Расширеный язык в рамках HTML. Используется для хранения и и передачи данных между компьютерами через web.

CSS3 - Cascading Style Sheets - Формальный язык описания внещнего вида страницы и ее элементов написаный посредством языка разметки HTML(XHTML). Используется в качестве инструмента для работы с дизайном отображаемой страницы с помощью цветов, шрифтов, стилей, расположения отдельных блоков и других аспектов внешнего вида.

JavaScript - Созданый Бренданом Эйхом универсальный язык программирования широко спектра использования. Язык является объекто\аспектно\событийно-ориентированным, императивным и функциональным. Основной язык для web разработки и реализован на основе языка C++.
